Explanation
This program determines the distribution of letter grades given some number
grades. The letter grade is found according to the following conversion rules:

A 90 – 100
B 80 – 89.99
C 70 – 79.99
D 60 – 69.99
F < 60

Given the following input

23
78
99
54
67
84
32
12
67
85
45
35

the output is as follows:

There is a student with the letter grade A.
There are 2 students with the letter grade B.
There is a student with the letter grade C.
There are 2 students with the letter grade D.
There are 6 students with the letter grade F.
